# Taskery

Taskery is a task tracking system created for learning purposes. 

## Used technologies 

- actix-web
- diesel
- gRPC(tonic)
- graphql
- docker
- redis
- postgres

## How to test

1. docker-compose up --build
2. go to http://localhost:5050/
3. play
